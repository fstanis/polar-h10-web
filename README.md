# polar-h10-web

Library for using the
[Polar H10](https://www.polar.com/en/sensors/h10-heart-rate-sensor) chest-strap
sensor from the browser, over
[Web Bluetooth API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Bluetooth_API).
In addition to real-time heart rate, it includes ECG and accelerometer and
creating and reading exercise recordings.

This library is an independent hand-port of the official
[Polar BLE SDK](https://github.com/polarofficial/polar-ble-sdk) and is not
affiliated with, authorized or endorsed by Polar Electro Oy.

## Demo

[Click here for a working demo](https://polar-h10-demo.apps.stanis.me/).

## Install

```sh
npm install polar-h10-web
```
