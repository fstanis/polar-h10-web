polar-h10-web sample

polar-h10-web talks to the Polar H10 chest strap over the Web Bluetooth API. This page walks through the basics: connect to a strap, then stream heart rate, raw ECG, and accelerometer data. Each button runs the snippet shown below it; results appear in the live output at the bottom.

Connect

Open the browser's device chooser (this requires a user gesture, hence the button), connect, and read the device info. All other snippets use the device this one creates.

JavaScript Snippet

Heart rate

Streams are async iterables — consume them with for await and stop them by aborting the AbortSignal passed at start. Heart rate delivers one sample per second with beat-to-beat RR intervals.

JavaScript Snippet

ECG

Raw ECG at a fixed 130 Hz, in microvolts. The snippet logs one sample per second.

JavaScript Snippet

Accelerometer

Acceleration in milli-G on three axes. Sample rate (25/50/100/200 Hz) and range (2/4/8 G) are chosen at start. The snippet logs one sample per second.
